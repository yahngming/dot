# dotfiles

### post install:
```sh
sudo pacman -S git && mkdir -p ~/.local/src && cd ~/.local/src && git clone https://github.com/yahngming/dot && bash ~/.local/src/dot/.local/bin/post
```

```sh
curl -Lo ~/.config/clash/config.yaml https://path/to/clash
```
